# Lidar-ranging-technology-and-algorithms-based-on-frequency-modulated-continuous-wave

一: Project Overview Undergraduate Design. The project uses OPTISYSTEM to create virtual laser ranging data, and uses MATLAB to extract the beat frequency signal to obtain the ranging distance. 

二: Experimental software: OPTISYSTEM 15.0, MATLAB 

三: Frequency signal extraction algorithm: Energy centroid method, Local interpolation method


一：项目概述 毕业设计，项目使用OPTISYSTEM创建虚拟激光测距数据，使用MATLAB提取拍频信号得到测距距离。

二：实验软件 OPTISYSTEM15.0、MATLAB

三：拍频信号提取算法：能量重心法、局部插值法
